== Description ==

This is an addon of Amazon S3 Integration allowing to add todo tasks in custom post types and taxonomies such as WooCommerce Product, Order, Category and Tags.

= Features =

- CUSTOM POST TYPES - Add Todo tasks in any Custom Post Type available such as WooCommerce product, order or coupon.
- CUSTOM TAXONOMIES - Add Todo tasks in any Custom Taxonomy available such as WooCommerce product category or tag.

More information about [Amazon S3 Integration](https://code.recuweb.com/download/aws-s3-integration/)

== Installation ==

= Minimum Requirements =

* Wordpress 4.6 or higher.
* Amazon S3 Integration 3.0.0 or higher.

= Manual installation =

The manual installation method involves downloading my plugin and uploading it to your webserver via your favourite FTP application.

1. Download the plugin file to your computer and unzip it
2. Using an FTP program, or your hosting control panel, upload the unzipped plugin folder to your WordPress installation's wp-content/plugins/ directory.
3. Activate the plugin from the Plugins menu within the WordPress admin.
